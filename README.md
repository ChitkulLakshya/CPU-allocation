# CPU Allocation / Paging Simulation

A simple web-based simulator for **paging** in Operating Systems.

## Features

- Simulates **Simple Paging**
- Simulates **Demand Paging**
- Uses **9 memory frames**
- Demonstrates **Process A with 3 pages**
- Shows the **page table** and **free frame list**
- Includes a visual comparison of simple paging vs. demand paging

## How to Run

1. Open `index.html` in a browser.
2. Choose **Simple Paging** or **Demand Paging**.
3. Click **Simulate Allocation** or **Request Page** to see the memory update.

## Files

- `index.html` — UI structure
- `style.css` — styling and layout
- `script.js` — paging simulation logic

## Notes

This project is designed as a visual learning aid for paging concepts in Operating Systems.
